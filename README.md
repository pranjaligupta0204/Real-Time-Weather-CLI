# ☁️ Real-Time Weather App (Python)

A simple Python project that fetches live weather data using the "WeatherAPI" service.  
The user enters a city name, and the program returns real-time temperature, humidity, wind speed, and weather conditions.

---

## Description
This project demonstrates how to interact with an external weather API, parse JSON data, and display the results in a clean, readable format. Helps in understanding how APIs work in real applications.

---

## Things Used
- Python  
- WeatherAPI (for weather data)  
- Requests library  
- JSON parsing  
- Simple CLI input/output  

---

## Learnings
- How to call a REST API using Python  
- Handling API responses and error conditions  
- Extracting useful information from JSON data  
- Creating a clean, interactive command-line program  
- Using real-time data in a Python application
